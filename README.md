# Purushotham.R - Wildlife Photography Portfolio

A premium, glassmorphism-inspired portfolio website for wildlife photography to showcase a curated collection of birds, mammals, landscapes, macro, and street photography.

## Features
- **Modern UI/UX**: Ultra-premium aesthetic featuring frosted glass effects (glassmorphism), sophisticated fluid typography, and dynamic CSS animations.
- **Responsive Layout**: Designed to look and function perfectly across mobile, tablet, and desktop devices.
- **Dynamic Image Gallery**: A sleek, filterable masonry-style grid organized by wildlife categories (Birds, Mammals, Landscape, Macro, Street).
- **Immersive Fullscreen Modal**: Offers a detailed view of high-resolution prints, complete with captions, active hashtags, timestamps, localized like counts, and a direct link to the original Instagram posts.
- **Advanced Design System**: Built with modular vanilla CSS featuring complex gradients, ambient glowing orbs, elegant layout shifts, and CSS custom variables for easy theming.
- **Enhanced Contact Section**: Premium quick-connect channels for Instagram, LinkedIn, Email, and WhatsApp.

## Technologies Used
- HTML5 (Semantic Structure & Accessibility via ARIA attributes)
- CSS3 (Vanilla Web Design System, Flexbox, Grid, Transforms, and Backdrop-Filters)
- JavaScript (Vanilla DOM Manipulation, Event Handling, and Mock Data Fetching)



## File Structure
- `index.html`: The core semantic markup, layout scaffolding, and SEO meta tags.
- `styles.css`: The comprehensive design system, ensuring consistent visually stunning effects across all components.
- `script.js`: Handles interactivity, filtering logic, simulated mock API for the image gallery, and the fullscreen modal behavior.
